安装

每个工具都提供 macOS 和 Windows 两个直接下载脚本，默认使用国内 npm 镜像。

1

点击 CCSwitch、Gemini CLI、Codex 或 Claude Code 下面对应系统的下载按钮。

2

下载后得到单个安装脚本，不需要解压 ZIP。

3

macOS 运行 *-macos.command；Windows 运行 *-windows.ps1。
